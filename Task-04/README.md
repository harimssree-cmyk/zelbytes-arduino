## Mounting Notes

- HC-SR04 mounted horizontally on the test bench.
- Sensor height: 15 cm above the table surface.
- Target used: flat cardboard sheet.
- Sensor faced the target directly (approximately 90° angle).
- Reliable measurement range observed: 5 cm to 250 cm.
- Blind zone observed below 2 cm.
- Measurements became unstable when the target was angled.
- Reflections from nearby objects occasionally affected readings.
