\# Automated Grow Bench v1



\## Objective



Automatic irrigation based on soil moisture.



\## Hardware Used



\- ESP32

\- Soil moisture sensor

\- Relay module

\- Water pump

\- Push button



\## Features



\- Auto irrigation

\- Manual override

\- Dashboard monitoring

\- Pump control



\## Serial Commands



STATUS



FORCE\_ON



FORCE\_OFF



\## Testing



\- Dry soil → Pump ON

\- Wet soil → Pump OFF
